# Flask TODO アプリ

Flaskで作成したタスク管理Webアプリです。

## 使用技術
- Python（Flask）
- HTML / CSS / JavaScript
- PostgreSQL
- Git / GitHub

## 機能一覧
- タスクの登録 / 編集 / 削除（CRUD）
- ステータス管理（未着手 / 進行中 / 完了）
- ステータスごとに絞り込み表示（フィルタ機能）
- 締切日による昇順・降順のソート機能
- タスク一覧をCSV形式でエクスポートする機能
